# Object Detection and Tracking

A simple AI-based project that detects objects in real-time using a webcam. Built with Python, OpenCV, and YOLO, it identifies objects and displays bounding boxes with labels for each detected item.

## Technologies Used

* Python
* OpenCV
* YOLOv3
* NumPy

## Run

```bash
pip install -r requirements.txt
python main.py
```
